# PetSell_App

---

### 프로젝트 설명
> 애완 동물 분양 리스트 및 애완 물품을 보여주는 어플을 개발하였습니다. <br>
> 팀원 2명과 함께 1개월의 시간을 갖고 진행하였습니다.

---

### 개발 도구
> + Android Studio
> + java

---

### PetSell_App 의 기능

1. 로그인, 회원가입 기능
2. 애완 동물 분양 확인 기능
3. 애완용품 리스트 및 구매 기능
4. 애완 동물 레시피 기능
